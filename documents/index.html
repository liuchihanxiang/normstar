    # 1. NsForm 表单组件

    # 1.1 NSForm 配置项

    参数 说明 类型 可选值 默认值
    v-model 表单数据模型 必填 object — {}
    form-list 配置表单数据(具体可查看 formItem 参数配置) object — {}
    is-column-mixins 表单配置数据类型是否为混入类型（表格查询表单和普通表单公用一个配置数据时，该属性为 true，会通过相应的字段配置其属于查询表单还是普通表单） boolean — false
    label-width 表单域标签的宽度，作为 Form 直接子元素的 form-item 会继承该值 number — 80
    disabled 是否禁用该表单内的所有组件。若设置为 true，则表单内组件上的 disabled 属性不再生效 boolean — false
    size 用于控制该表单内组件的尺寸 string medium / small / mini medium
    status-icon 是否在输入框中显示校验结果反馈图标 boolean — true
    label-position 表单域标签的位置，如果值为 left 或者 right 时，则需要设置 label-width string right/left/top right

    # 1.2 form-list 配置项

    参数 是否必填 说明 类型 可选值 默认值
    type 否 表单类型 string 具体参数查看 type 配置项 text
    prop 是 表单域 model 字段（与后台字段对应） string — —
    label 是 标签文本（在查询表单里会转化为 placeholder） string — —
    rules 否 表单验证规则（具体规则查看 elementUI 表单验证规则） object — —
    default 否 表单域默认值 * — —
    dicData 是 当表单为 select redio checkbox 时的数据字典 array — []
    valueKey 否 当表单为 select 时,选项的值对应字段 string — value
    labelKey 否 当表单为 select 时,选项的标签文本对应字段 string — label
    class 否 当前表单域类名（更改样式使用） string — —
    colClass 否 当前表单域类名外层类名（更改样式使用） string — —
    slotName 否 一些特殊表单（type 为 slot 时），需要以插槽的形式写入表单，插槽名称 string — —
    form 否 一些特殊表单（type 为 slot 时），内有多个表单时 对应的 model object — —
    show 否 是否渲染改表单（内部使用 v-if 实现的），一些复杂表单，控制有无该表单 bpplean — —

    # 1.3 表单类型可选项

    • text 普通文本框,也是表单默认值
    • textarea 文本域
    • checkbox 多选框
    • radio 单选框
    • select 下拉框（默认单选，配置 multiple:true 即可实现多选）
    • timePicker 时间选择器（配置 is-range:true，即可实现时间选择范围）
    • year/month/date/dates/ week/datetime/datetimerange/daterange 日期选择器
    • number 计数器
    • switch 开关
    • slot 插槽类型 此时必须配置 slotName 字段
    • infoTextarea 编辑器

    # 2. NSTable 表格组件

    # 2.1 NSTable 配置项

    参数 说明 类型 可选值 默认值
    url 表格数据请求地址 string — —
    form-list 配置表格上方查询表单数据（具体查看 NsForm 配置说明） object — {}
    btn-list 配置表格上方操作按钮 array — []
    columns 配置表格列数据（具体查看 columns 配置参数） array — []
    operations 配置表格右侧操作 list array — []
    query-params 表格请求数据前参数处理 function — —
    tree-table 是否为树形表格 boolean — false
    show-search-form 是否显示表格上方查询表单 boolean — true
    operations-only-show-icon 表格操作是否只显示图标(false 时将只展示操作文字) boolean — true
    operations-auto-dropdown 操作列过多时，是否自动转为下拉形式 boolean — true
    operations-auto-dropdown-max-num 表格操作大于等于该数值时会展示下拉，隐藏一部分操作 number — 4
    list-field 返回数据位置 string — data.data
    total-field 返回的数据总数 的数据位置 string — data.total
    page-list 表格分页 每页可选配置项 string — [10, 20, 50, 100]
    pagination-layout 配置分页显示项 string 'total, prev, pager, next, jumper, sizes' 'total, prev, pager, next, jumper, sizes'
    pagination 是否分页 number — true

    # 2.2 columns 配置项

    参数 说明 类型 可选值 默认值
    prop 对应列内容的字段名 string — —
    label 显示的标题 string — —
    formatter 用来格式化内容 Function(row, column, cellValue, index) — —
    type 对应列的类型。如果设置了 selection 则显示多选框；如果设置了 index 则显示该行的索引（从 1 开始计算）；如果设置了 radio 则显示为单选 string selection/index/radio —
    slotName 插槽名称 （特殊列需采用插槽形式） string — —
    sortable 是否开启排序 boolean — —
    show-overflow-tooltip 当内容过长被隐藏时显示 tooltip Boolean — —
    width 对应列的宽度 string — —
