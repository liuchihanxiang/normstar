    # 1. 项目基础配置

    # 1.1 环境配置

    1. node 下载地址 按照默认配置安装即可 安装完node 会同步安装npm，这时就可以用npm 命令安装各种包
    2. webpack 全局安装 cmd执行命令
      npm i -g webpack
    3. node-sass 全局安装 cmd执行命令
      npm i -g node-sass
    4. vue-cli 全局安装 cmd执行命令
      npm install -g @vue/cli

    # 1.2 编辑器

    编辑器推荐使用 vscode 下载地址 (其他编辑器也可以，这里详细说明vscode 相关配置及其常用插件)

    # 1.2.1 常用插件

    • Vetur VSCode上开发Vue必备工具
    • Eslint 代码风格检测
    • Prettier-Code formatter 格式化插件
    • js-css-html Formatter 格式化插件
    • HTML Snippets html代码片段 提高开发效率
    • Document this 多行代码注释插件 只能在js中 函数上方使用 ctrl+alt+D连续按两次
    • One Dark Pro 主题插件
    • vscode-element-helper elementUI代码提示插件

    # 2. 项目结构说明

    |———— static                           // 纯静态资源，不会被wabpack构建
|———— node_modules                     //存放项目依赖包文件夹
|———— public                           //
|———— tests                            //测试
|———— src                              //主要开发文件夹
|     |———— assets                    //静态资源 会被wabpack构建
|     |———— components                 //存放项目中用到的组件
|     |———— http                       //基于axios封装的http请求
|     |———— mixins                     //混入
|     |———— directive                  //指令
|     |———— utils                      //工具类
|     |———— store                      //状态管理
|     |     |———— modules              //模块化状态
|     |     |____ getter.js
|     |———— styles                      //项目css
|     |     |———— common.scss           //公共css
|     |     |____ mixin.scss            //常用的mixin
|     |     |____ variables.scss        //常用的scss变量
|     |———— router                     //项目路由
|     |     |———— constant-route.js    //静态路由
|     |     |____ dynamic-route.js     //动态路由
|     |———— api                        //项目中所有请求管理
|     |     |———— user.js              //用户模块api
|     |     .
|     |———— config                     //系统配置文件
|     |     |———— env.js               //项目基础变量配置
|     |     |____ http-code.js         //项目中http状态码  
|     |
|     |———— const                      //项目中一些复用率特别高的常量 全部放此处便于管理
|     |     |———— dic-data.js          //前端数据字典
|     |     |———— icon.js              //常用图标名称 （主要是table表格操作栏）                               
|     |———— page                      //项目中一些独立页面
|     |———— views                     //项目中main主要页面
|
|
|———— App.vue                         //根组件
|———— store.js  
|———— router.js                            
|———— .eslintrc.js   //eslint代码检查 配置文件
|———— vue.config.js webpack配置文件（端口号 跨域等等）
